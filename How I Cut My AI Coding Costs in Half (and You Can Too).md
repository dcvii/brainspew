I was juggling Claude Code, Cursor, and Codex — switching between them depending on the task, telling myself it was worth it. Then I did the math. It wasn’t.

So I started looking for a smarter setup, and I found one that costs me $10/month and still lets me use Claude, GPT-4, and Gemini. Here’s what I’m running now

### Meet OpenCode

[OpenCode](https://opencode.ai/) is a terminal-based AI coding assistant — think Claude Code vibes, but with one thing I immediately liked better: the side panel.

While you’re working, it shows you exactly what’s happening — context usage, which files have been modified, what the model is doing. It sounds small, but when you’re deep in a session, that visibility is genuinely useful. No more guessing how close you are to the context limit or which files got touched.

### The actual money-saving trick: GitHub Copilot

Here’s the part that surprised me. GitHub Copilot isn’t just a model — for $10/month, you get access to a bunch of major models from Anthropic, OpenAI, and Google, all under one subscription.

![](https://miro.medium.com/v2/resize:fit:1400/format:webp/1*mq3WEjroe1Ka_9LN9XQYTw.png)

OpenCode plugs right into it. So instead of paying for separate Claude and OpenAI subscriptions, I just route everything through Copilot and pick whichever model fits the task. Need strong reasoning? Swap to one. Need fast iteration? Swap to another. Same project, same terminal, no extra cost.

*(One thing worth noting: the Claude Code subscription no longer works with OpenCode, so if that was your plan — Copilot is the move.)*

### Who is this for?

Honestly, anyone who’s been curious about AI coding tools but doesn’t want to commit to $40+ a month across multiple subscriptions. If you’re just getting started or want to experiment without burning money, this is a solid entry point.

Give it a try and let me know how it goes — genuinely curious if others are finding the same value I did.
